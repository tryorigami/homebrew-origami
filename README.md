# Homebrew Origami

Homebrew tap for [Origami](https://tryorigami.app/) — a project-oriented terminal manager for macOS.

## Install

```bash
brew install --cask ricardobucho/origami/origami
```

Or tap first:

```bash
brew tap ricardobucho/origami
brew install --cask origami
```

## Update

Origami self-updates via its built-in updater. `brew upgrade` works too.

## Uninstall

```bash
brew uninstall --cask origami
```
