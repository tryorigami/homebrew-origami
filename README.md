# Homebrew Origami

Homebrew tap for [Origami](https://tryorigami.app/) — a workspace-oriented terminal manager.

## Install

```bash
brew install --cask tryorigami/origami/origami
```

Or tap first:

```bash
brew tap tryorigami/origami
brew install --cask origami
```

## Update

Origami self-updates via its built-in updater.
`brew upgrade` works too.

## Uninstall

```bash
brew uninstall --cask origami
```
